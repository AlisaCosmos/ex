## Сайт для веб-агентства

### На сайте реализовано:

- адаптивная кроссбразуерная верстка по макету из Figma посадочной страницы сайта;
- поэкранный адаптивный скролл страницы;
- анимации.

http://test3.alisawebdev.com/
